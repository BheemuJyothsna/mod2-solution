HTML


HTML (Hypertext Markup Language) is the code that is used to structure a web page and its content. For example, content could be structured within a set of paragraphs, a list of bulleted points, or using images and data tables and also to links to access pages.

CSS


CSS stands for Cascading Style Sheets, and it's used to add style to a web page by dictating how a site is displayed on a browser. CSS is unique in that it doesn't create any new elements, like HTML or JavaScript. Instead, it's a language used to style HTML elements.

JAVASCRIPT


JavaScript is used to develop interactive web applications. JavaScript can power featured like interactive images, carousels, and forms. The language like Node. js used to power the mechanics behind a web page, such as form processing and payments.
